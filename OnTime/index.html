OnTime - Fleet Management and Bus Ticketing System

Akshaya Sivakumar Karunambika , Kaushik Prasanth Vijay Baskaran, Pallavi Singh

#Demos #Application Details #Tech Stack

     A Web Application for fleet management and bus ticketing.

Features

For Users

  • Create Profile to manage bookings
  • Search,View Trips
  • Book Tickets Online

For Admins

  • Add New Destinations
  • Create Routes and add trips
  • Assign Trips to drivers
  • Create Driver Profile
  • Add New Bus to fleet and modify the seat layout
